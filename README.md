# Fast Collage Creator

> [!WARNING]
> This has been coded mostly by AI and not yet thoroughly checked. Use at your own risk. Contributions to improve the code are very welcome.

Fast Collage Creator is a Python application for quickly generating image collages. It provides a graphical interface to select images, arrange them, and export collages efficiently. The project is designed for speed and ease of use, supporting various image formats and customizable layouts.

## Features
- Customizable collage layouts
- Easy-to-use graphical interface

## Running Instructions

1. **Install dependencies:**
   - Make sure you have Python 3.8+ installed.
   - Install required packages:
     ```bash
     pip install -r requirements.txt
     ```

2. **Run the application:**
   - Start the main window:
     ```bash
     python main.py
     ```
